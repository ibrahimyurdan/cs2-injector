This directory needs an icon.ico file for the application.

For a complete build, please create an icon.ico file in this directory. The icon should be:
- 32x32 pixels (minimum)
- Multiple resolutions supported (16x16, 32x32, 48x48, 256x256)
- Include both 32-bit and 8-bit color depth versions

You can create this using an icon editor or convert from an existing image file.
